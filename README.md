# Шуховская башня — интерактивная 3D-модель

Параметрическая реконструкция гиперболоидной радиобашни В. Г. Шухова
на Шаболовке (Москва, 1920–1922) на three.js. Геометрия строится в браузере
из конструктивных параметров, а не загружается готовым мешем.

**Демо:** https://sashamind.github.io/shukhov-tower/

## Конструкция

| Параметр | Значение |
|---|---|
| Высота без траверс | 148 м |
| Полная высота | 160 м (траверсы и флагшток — 12 м) |
| Секций | 6 × 25 м, каждая — однополостный гиперболоид |
| Стержней в секции | 48 у четырёх нижних, 24 у двух верхних |
| Кольца | от ⌀ 40,3 м у фундамента до ⌀ 3,2 м наверху |
| Масса | 240 т |

Каждая секция «соткана» из двух встречных семейств **прямых** стержней —
криволинейная поверхность гиперболоида целиком образована прямыми линиями.
Ползунок «Закрутка» масштабирует углы: на 0 % башня вырождается в набор конусов.

## Источники данных

Числа стержней и диаметры колец — по результатам лазерного сканирования башни
(Leonov A. V. et al. *Laser scanning and 3D modeling of the Shukhov hyperboloid
tower in Moscow* // Journal of Cultural Heritage, 2015, vol. 16, pp. 551–559).

Углы закрутки секций (26° → 70°, растут кверху) — **реконструкция**: выведены
из известных длин стержней (20–27 м), высоты секции и радиусов колец
через геометрию гиперболоида. Точной опубликованной таблицы углов найти не удалось.

## Структура

```
index.html          — вся модель, один файл, зависимость только three.js с CDN
shukhov-tower.obj   — экспорт с разбивкой на 9 именованных объектов
gen_obj.py          — генератор OBJ (та же параметрика, что в браузере)
```

## Blender

`File → Import → Wavefront (.obj)`, настройки по умолчанию — Y-up развернётся
в Z-up автоматически. Модель в реальном масштабе (метры), камеру ставить
метрах в 300. Объекты разбиты по секциям — можно красить и анимировать
по отдельности, например воспроизвести телескопический монтаж.

Кнопка «Скачать .obj» на странице выгружает модель с текущим положением
ползунка закрутки — одним мешем, без разбивки.

Перегенерировать посекционный OBJ:

```bash
python3 gen_obj.py
```
